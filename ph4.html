Cookie Run

ในภาพนี้ นักเรียนชายคนหนึ่งนั้นก็คือผมที่กลับมาจากกินข้าวจากภาพที่แล้วนั่งอยู่ที่โต๊ะในห้องเรียน ผมกำลังถือแท็บเล็ตด้วยสองมือและกำลังตั้งใจเล่นเกมบนหน้าจอของแท็บเล็ตชื่อเกม Cookie Run หน้าจอแสดงภาพเกมที่มีลักษณะเป็นการ์ตูนน่ารักๆ มีองค์ประกอบของเกมที่เป็นกราฟิกสีสันสดใส ผมสวมเสื้อเชิ้ตสีขาวและสเวตเตอร์สีดำ เช่นเดียวกับในภาพก่อนหน้านี้ใช่ครับวันเดียวกันจะไม่เหมือนกันได้ไง ผมมีท่าทางมุ่งมั่นในการเล่นเกมมากกว่าเรื่องเรียนอีก
